О проекте GameRequirements

GameRequirements — это сервис для проверки совместимости игр с вашим железом. Мы помогаем геймерам понять, какие игры запустятся на их компьютере и с каким FPS.

Возможности

  • Проверка совместимости игр с вашим железом
  • Предварительный расчет FPS для разных настроек графики
  • База данных популярных игр
  • Рекомендации по апгрейду
